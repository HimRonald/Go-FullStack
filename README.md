# Go FullStack Todo Application

This is a simple Todo application built using Go and the Fiber web framework. The application allows you to create and manage a list of todos.

## Getting Started

### Prerequisites

- Go 1.23.5 or later
- Git

### Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/HimRonald/Go-FullStack.git
    cd Go-FullStack
    ```

2. Install the dependencies:

    ```sh
    go mod tidy
    ```

### Running the Application

To run the application, use the following command:

```sh
go run main.go
